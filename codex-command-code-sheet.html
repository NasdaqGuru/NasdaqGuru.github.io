Quick Paste Commands

Codex Command Code Sheet

Use these short codes when you want me to edit the report suite without writing a long explanation.

How to use: paste the code, then add the section name if needed. Example: /exec Section XIII or /popup pipeline charts.

Most Useful Codes

/exec Executive-ready

Make the wording polished for VP/President review.

/simple Simplify

Reduce overwhelm, shorten text, and make the layout easier to scan.

/clean Clean design

Fix clutter, spacing, odd colors, crowded charts, or messy cards.

/popup Move to popup

Keep the page clean and put large charts/tables behind a click.

/stay Keep reader here

Remove links that send the reader away too early.

/explain Explain metric

Clarify a number, percentage, label, or chart title.

/sync Apply everywhere

Make the same fix across the whole report suite.

/decide Use best judgment

Let Codex choose the best strategy and implement it.
